Uma visão didática do fluxo de cadastro e monetização.

FASE 1: Cadastro e Boas-Vindas (Custo Zero)

App React

Supabase

Edge Function

1. Início no App

Usuário preenche o formulário de cadastro. O hook useAuth normaliza o telefone e chama o método de signUp do Supabase.

2. Mágica no Banco

Um novo usuário é criado. Triggers no PostgreSQL definem o período de trial e disparam um webhook de boas-vindas.

3. Olá, Mundo!

A Edge Function send-welcome-webhook recebe os dados e os envia para uma API externa (iAutomatize) para iniciar a comunicação.

FASE 2: Upgrade e Acesso Premium (Monetização)

Ação no App

Stripe Checkout

Supabase

4. Decisão de Compra

Usuário clica em 'Assinar'. A Edge Function create-checkout-session gera um link de pagamento seguro do Stripe.

5. Pagamento Confirmado

O Stripe processa o pagamento e notifica o sistema via webhook. A Edge Function handle-payment-success é acionada.

6. Acesso Liberado!

A função atualiza o status do usuário para 'ativo' no banco de dados. O app reflete a mudança e libera as funcionalidades premium.
